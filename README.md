## Usage

Please execute following command
````
Linux
% cd $HOME
% git clone https://github.com/hosoya/dotfiles.git
% cd dotfiles
% source setup.sh
````

````
Windows7 (emacs only)
cd \Users\username
git clone https://github.com/hosoya/dotfiles.git
mklink /d e.macs.d \Users\username\dotfiles\.emacs.d
Note: cmd prompt: Run as adminstrator
````
